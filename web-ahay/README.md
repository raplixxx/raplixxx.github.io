# Web Ahay - AI Chatbot with vesta.ai

Web Ahay adalah aplikasi chatbot AI cerdas yang menggunakan GPT-4o-mini melalui SumoPod API.

## Fitur Utama
- 🤖 AI Chat dengan GPT-4o-mini
- 🔍 Web Search Real-time (Tavily API)
- 🖼️ Analisis Gambar
- 🧠 Memory 20 Pesan Terakhir
- 📱 Responsive Design
- 🔗 Share Chat Publik
- 🔐 Google Authentication

## Teknologi
- Frontend: HTML5, CSS3, Vanilla JavaScript
- Backend: Firebase (Auth, Firestore, Hosting)
- AI: SumoPod API (GPT-4o-mini)
- Search: Tavily API
- DNS/CDN: Cloudflare

## Instalasi
1. Clone repository
2. Deploy ke Firebase Hosting
3. Konfigurasi Cloudflare DNS

## Deployment
```bash
firebase deploy
